## Windows 资源管理器的地址栏
|命令|功能|
|-|-|
|cmd|在当前路径打开命令提示符|
|powershell|在当前路径打开 PowerShell|
|notepad|打开文本文件|
|shell:startup|打开“启动”文件夹|
|shell:sendto|编辑右键菜单的“发送到”选项|
|shell:appsfolder|查看所有已安装的应用|
|..|返回上一级目录|
|control|打开控制面板|
|mstsc|快速调出远程桌面连接|
|appwiz.cpl|直接进入“卸载或更改程序”界面|
|sysdm.cpl|修改环境变量|
|ncpa.cpl|直接打开“网络连接”|
|snippingtool|启动截图工具|
|mrt|启动 Windows 恶意软件删除工具|
|resmon|比任务管理器更详细|
|perfmon|打开“性能监视器”|
