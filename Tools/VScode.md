### 打开当前目录作为工作区
`cmd /c code . `

### VScode快捷键
|功能|command|
|-|-|
|打开命令面板|Ctrl + Shift + P|
|打开键盘快捷键|Ctrl + K, Ctrl + S|
|打开设置|Ctrl + ,|
|打开终端|	Ctrl + `|
|快速打开，转到文件|	Ctrl + P|
|格式化代码|Shift + Alt + F|
|注释行|Ctrl + /|
|多行注释|Shift + Alt + A|
|复制当前行|Alt + Shift + Down|
|删除当前行|Ctrl + Shift + K|
|移动当前行|Alt + Up/Down	|
|选中当前行|Ctrl + L|
|选中所有匹配内容|Ctrl + Shift + L|
|选中下一个匹配|Ctrl + D|
|放大/缩小|Ctrl + = / -|
|切换侧边栏可见性|Ctrl + B|
|显示资源管理器|Ctrl + Shift + E|
